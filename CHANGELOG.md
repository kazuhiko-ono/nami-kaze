# 全国の波と風 - 改修履歴

## 概要

全国のビーチの風・波・潮位をリアルタイム表示するWebアプリ「全国の波と風」の開発・改修履歴です。

---

## 主要な機能追加

### アメダス実測データ対応（最新）
- 現況の風データをアメダス（JMA）実測値に切り替え
- 最寄りのアメダス観測点を自動検出し、風速・風向・気温・湿度を表示
- アメダス取得失敗時は Open-Meteo にフォールバック

### マルチモデル風速予測
- 風速グラフに GFS / ICON / GEM / ECMWF の4モデル予測線を追加
- 色分け凡例付きでモデル間の比較が可能

### 潮位データの地点別対応
- 潮位APIが江ノ島にハードコードされていた問題を修正
- 各スポットに tide736.net の最寄り観測点コードを割り当て
- スポット名から潮位観測点を自動解決（古いURLでも正しく動作）

### 波高グラフ
- Open-Meteo Marine API の波高データを Chart.js 折れ線グラフで表示
- 波高に応じた色分け（0.5m以下: 薄青、0.5〜1.5m: 緑、1.5m以上: オレンジ）
- 風速・潮位グラフと同じ時間軸・幅でスクロール連動

### 5日間予報
- 時間別予報・風速グラフ・潮位グラフの表示期間を48時間から120時間（5日間）に拡張

### お気に入り機能
- マップ画面: ポップアップに☆/★トグル、お気に入りバー（最大3地点）
- コンディション画面: ヘッダーに☆/★お気に入りトグル
- トースト通知によるフィードバック（追加・削除・上限）

### 初回ガイド & メール登録
- 初回訪問時にガイドオーバーレイを表示
- メール収集フォーム（将来の通知機能向け）

### 全国マップ
- Leaflet.js による全国マップページ（エリア別色分け）
- 地点をクリックしてコンディション画面へ遷移

## UI改善

- 時間別予報・風速グラフ・潮位グラフの時間軸を縦に揃え
- 地図のズームレベルを調整（15→12）し、海岸線と陸地が見えるように改善
- お気に入りのフィードバックを alert からトースト通知に変更
- OGP画像を SVG から PNG に変更（SNSでの文字化け対応）

## データソース変更

- 気象モデルを GFS 指定 → Open-Meteo デフォルト（観測ベース）→ アメダス実測 と段階的に改善
- クレジット表示を追加

## 座標修正

- 全ポイントの座標をビーチ正面の海上（沖合約300m）に再設定
- 由比ヶ浜、逗子、津久井浜、磯ノ浦、生見、木崎浜の座標を個別修正
